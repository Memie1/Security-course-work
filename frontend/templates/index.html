What this prototype includes

Search

Users can search products by name or category.

Product Details

Each product opens into a detail modal with price and description.

Purchase Flow

Users can add products to cart and see a running total.

Reviews

Registered-user style review form with basic text formatting buttons and image upload field.

Access levels

User, seller and admin areas

Customer account

Normal users can create an account, log in, buy items and submit reviews after purchase.

  • Create account
  • Log in and log out
  • Purchase products
  • Leave reviews

Seller account

Regular users can upgrade to a seller role and manage their own listings.

  • Upgrade account role
  • List new products
  • Edit existing products
  • View transaction history

Admin account

Admin capabilities are represented as secured management features in the prototype design.

  • Manage all users
  • Create and manage listings
  • Monitor logs and activity
  • Review platform behaviour

Logging and analytics

Activity tracking justification

This prototype includes simulated logging ideas for page views, product clicks, cart actions and purchases. These are the most useful events for security monitoring, seller reporting and platform improvement without collecting excessive personal data.

Page views

Shows general user activity and helps detect unusual browsing spikes.

Product interactions

Tracks popular items and suspicious repeated access behaviour.

Purchases

Supports transaction records, fraud checks and seller history.

Role actions

Useful for admin auditing, seller listing changes and accountability.
